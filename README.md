# Sangeetha Divya - Portfolio

A beautiful, modern, and interactive portfolio website showcasing my journey as a Senior Software Engineer specializing in .NET backend development.

## ✨ Features

- **🎨 Modern Design**: Stunning animated gradient backgrounds and smooth transitions
- **📱 Fully Responsive**: Optimized for all device sizes with mobile-first design
- **🎯 Interactive Elements**:
  - Typewriter effect for name animation
  - Smooth scrolling navigation with progress indicator
  - Floating particle effects in hero section
  - Hover animations on cards and buttons
  - Scroll-triggered animations
- **🏆 Clickable Certifications**: Interactive certificates with modal preview and download options
- **📋 Comprehensive Sections**:
  - Hero section with floating animation
  - About me with professional contact details
  - Skills categorized with technology tags
  - Professional journey timeline
  - Featured projects showcase
  - Contact information

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Modern CSS with Grid, Flexbox, Animations, and CSS Variables
- **Typography**: Google Fonts (Poppins)
- **Icons**: Unicode emojis and custom CSS icons
- **Interactivity**: Vanilla JavaScript with Intersection Observer API

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Visit `http://localhost:8000`

## 📜 Certifications

The portfolio includes interactive certifications:
- **Azure Fundamentals (AZ-900)** - Microsoft
- **GitHub Copilot** - GitHub

Click on any certification to view details in a modal popup with download and preview options.

## 🎨 Customization

### Colors
The design uses a beautiful animated gradient background. To customize:
- Update the `background` property in `body` CSS rule
- Modify gradient colors in various elements

### Content
- Update personal information in `index.html`
- Modify skills, projects, and certifications
- Add your actual certificate files to enable real downloads

### Animations
- Adjust animation speeds in CSS `@keyframes`
- Modify particle count in `createParticles()` function
- Customize hover effects and transitions

## 📱 Responsive Design

The portfolio is fully responsive with breakpoints for:
- Desktop: > 768px
- Tablet: 768px - 480px
- Mobile: < 480px

## 🌟 Performance Features

- Optimized CSS animations
- Efficient JavaScript with event delegation
- Lazy loading of scroll animations
- Minimal external dependencies
